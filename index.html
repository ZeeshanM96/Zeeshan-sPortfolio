MontyHall App

This project combines C# for the backend and ReactJs for the frontend to create a Monty Hall simulation. The Monty Hall problem is a probability puzzle where a contestant selects one of three doors, one hiding a car and the others concealing goats. The API's purpose is to enable users to experiment with the Monty Hall problem's probabilities, emphasizing the outcomes based on switching doors or maintaining the original selection. This project employs .NET for the backend and integrates Swagger for API testing and documentation.

Real Time Face Recognition

This project involves real-time face detection and recognition through webcam video feed. It utilizes Python, OpenCV for video capture and visualization, the Google Cloud Vision API for face detection, and the face_recognition library for face recognition. The Python-based system captures webcam footage, applies face detection to locate faces in the video, and utilizes face_recognition to recognize and compare faces through their encodings..

Hollywood Gender Dialogue Analysis

In this project, we examined a dataset containing information about 1037 Hollywood movies and their lead roles. Our objective was to predict the lead role in a movie based on various features provided in the dataset, such as revenue, year, and the number of dialogues. We initially analyzed the data to gain insights and formulated assumptions for prediction. Subsequently, we implemented Logistic Regression and Random Forest models to validate our assumptions. We also introduced a naive classifier for comparison. Through Cross Validation, we assessed the accuracy of our models and identified one that yielded more precise predictions. This chosen model will be applied to a test dataset to evaluate its performance.

Crypto Data Analysis

Two methods are used: the ARIMA model for forecasting and linear regression for predicting BTC prices based on volume, with performance evaluated using accuracy metrics.
